# Estudos de Computação e Softwares
Este repositório tem como objetivo registrar e organizar todo o meu conhecimento teórico adquirido por meio de leituras e faculdade, relacionando-o com a arquitetura completa da computação. A proposta é consolidar conceitos fundamentais e servir como base de estudo e consulta contínua ao longo da minha jornada na área de tecnologia.
